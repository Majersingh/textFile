## Environment Variables

Before running the scripts, set the following environment variables:

```bash
export CORE_APPLE_PAYMENTS_API_ROOT=http://core-apple-payments.production-01.instant.paidy.net:80
export MERCHANT_MEMBER_ID=mem_VtTx-S4AAOEKNOYw
export ELASTICSEARCH_API_ROOT=http://your-elastic-url
export KYC_API_ROOT=http://your-kyc-url
```

Ensure you have the following installed:
